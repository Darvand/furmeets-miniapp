# FurMeets MiniApp

Telegram allow bots to display a web app in their context, allowing us to show anything we want using browser techniques: [FurMeets MiniApp](t.me/furmeets_bot/hub)
### Tech Stack
- ReactJS
- TypeScript
- Redux Tool Kit
- Vite
- @telegram-apps/sdk-react

## Infraestructure

For a free deployment infraestructure, this repository uses [Vercel](vercel.com)
